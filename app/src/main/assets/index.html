Introduction

Alien Simple Skin for Minecraft V1.0 is source code to make skin for minecraft Pocket Edition, used reward ads for unlock skin.

Technology Use:

  • Java Programming
  • Json Format
  • HTML Format
  • Android Studio 4.2.1
  • Sublime Text/Notepad++

Features:

  • It uses 5 advertisements with switch Ads system (Admob, Facebook Audience Network,Mopub, Applovin dan StartApp)
  • Open Bidding : Facebook Audience Network, Applovin, Mopub dan Admob
  • 2D and 3D Skin Viewer
  • Download and Install button with Intertitial ads
  • Reward ads for unlock skin
  • Banner on bottom layout
  • New Design Exit Dialog
  • The latest GDPR configuration
  • Redirect App Module to handle suspended application
  • More App Module with listview design
  • Data using by online and offline mode which is stored in json format
  • In App Update and In App Review Module
  • Switch ads and backup ads mode usage

Support Files:

  • Skin (*.png)
  • Json
